# Unitopia

Unitopia is a powerful and user-friendly unit conversion app built in SwiftUI for iOS. It allows users to seamlessly convert various types of units with ease. Designed with simplicity and efficiency in mind, Unitopia provides a clean and intuitive interface for quick and accurate conversions.

## What's New in 2.0

- **Favorites Redesigned**: Tap the star icon in the converter to save both the conversion result and the unit pair in a single action.
- **Conversion History**: Recent conversions are automatically saved and displayed on the Home tab.
- **Searchable Favorites**: Search across favorited conversions and unit pairs from the Favorites tab.
- **Segmented Favorites View**: Favorites are now organized into two segments — Conversions and Unit Pairs.
- **SwiftData Persistence**: History and favorites are persisted locally using SwiftData.
- **Haptic Feedback**: Custom Core Haptics pattern plays on reset.
- **Animated Welcome Screen**: New onboarding screen shown on first launch and after updates.

## Features

- **Comprehensive Unit Conversions**: Convert between Temperature, Distance, Mass, Time, Speed, and Volume.
- **Favorites**: Save unit pairs and conversions for quick access.
- **Conversion History**: Browse and re-open recent conversions from the Home tab.
- **Dark Mode Support**: Easily toggle between light and dark themes.
- **Offline**: All conversions happen on-device — no internet required.
- **Optimized for iOS**: Built with SwiftUI, ensuring a smooth and modern user interface.
